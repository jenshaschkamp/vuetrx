Deploying to Heroku using the Command Line

You can also deploy Vuetrx to Heroku using the command line:

  1. Clone the repository

    git clone https://github.com/ClearbridgeMobile/vuetrx

  2. Create a Heroku application

    cd vuetrx
heroku create

  3. Install Heroku Connect 

    heroku addons:create herokuconnect

  4. Deploy to Heroku

    git push heroku master

  5. Run the the application:

    heroku open

Installing a Local Version

You can also install Vuetrx on your local machine:

  1. Clone the repository

    git clone https://github.com/ClearbridgeMobile/vuetrx

  2. Install the server dependencies

    cd vuetrx
npm install

  3. Create a local database

  4. Start the server 

    node server

  5. Run the application

    • Open a browser and access the following URL: http://localhost:5000
